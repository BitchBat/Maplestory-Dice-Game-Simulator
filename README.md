# 🎲 진의 신비한 정원 주사위 시뮬레이터 (Jin's Mysterious Garden Simulator)

보드게임에서 주사위를 굴렸을 때의 **최종 도착 위치**와 **획득하는 코인(재화)의 양**을 미리 계산해주는 시뮬레이터입니다. 
복잡한 타일 효과(이동, 코인 획득, 미스터리 박스 등)를 자동으로 계산하여 최적의 플레이를 계획할 수 있도록 도와줍니다.

---

## 📥 프로그램 다운로드 방법 (GitHub가 처음이신 분들을 위해)

복잡한 설치 과정 없이, 실행 파일(`.exe`) 하나만 다운로드하면 바로 사용할 수 있습니다!

1. **릴리즈(Releases) 페이지 접속**
   * 현재 보고 계신 화면 우측에 있는 **[Releases]** 탭으로 이동하세요.

2. **최신 버전 다운로드**
   * 가장 위에 있는 최신 버전(Latest) 상자 안에서 **`Assets`** 라는 글씨를 클릭하여 목록을 펼칩니다.
   * 목록 중에서 **`.exe`** 로 끝나는 파일을 클릭하여 다운로드합니다.

3. **프로그램 실행**
   * 다운로드 받은 `.exe` 파일을 더블클릭하여 실행합니다.
   * 💡 **참고 (Windows 보안 알림이 뜰 경우)**: 
     개인이 만든 프로그램이라 Windows에서 "PC 보호" 파란색 창이 뜰 수 있습니다. 
     당황하지 마시고 **[추가 정보]** 글씨를 누른 후, 우측 하단에 생기는 **[실행]** 버튼을 누르시면 정상적으로 켜집니다.

---

## ✨ 주요 기능

* **주사위 시뮬레이션**: 현재 위치와 3개의 주사위 눈금을 입력하면 최종 도착 타일과 누적 코인을 즉시 계산합니다.

---

## 🚀 사용 방법

1. **현재 위치 설정**: 화면 좌측의 설정 패널에서 현재 내 캐릭터가 위치한 타일 번호를 선택합니다.
2. **주사위 눈금 입력**: 굴릴 예정이거나 굴린 주사위 3개의 값을 입력합니다.
3. **시뮬레이션 실행**: `시뮬레이션 실행` 버튼을 클릭합니다.
4. **결과 확인**: 우측 패널에서 주사위를 굴릴 때마다 거쳐가는 경로, 발동된 타일 효과, 그리고 최종적으로 획득한 총 코인량을 확인합니다.
